# Kinvova Gen3 in ROS
Kinova Kortex

## 使用方式
- 編譯 Package
    ```shell
    $ catkin_make
    ```
    若出現編譯失敗的情形，可以先再次嘗試 `catkin_make`。
- 清除編譯檔 
    ```shell
    $ catkin_make clean
    ```
- 執行
    ```shell
    $ rosrun kinova_test <executable file>
    ```
